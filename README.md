# ColorOSTool

增强 ColorOS(For OnePlus) 使用体验

## 当前功能：
- 解除只允许5个非白名单应用自启动的限制
- 解除启动器只能锁定5个后台应用的限制
- 移除apk安装前的安全验证
- 移除安装前，因版本相同、低版本等弹出的警告提示
- 自带的原生安装界面可选
- 移除锁屏时钟里的红色1
- 屏蔽游戏空间Root检测

## 注意事项
- 安装或更新模块后，请强行停止推荐HOOK的应用以生效。
- 切换模块功能开关后，请强行停止推荐HOOK的应用以生效。
- 当前仅在 ColorOS v11.2 For OnePlus 上测试过。
